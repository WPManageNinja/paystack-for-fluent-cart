Accept payments via Paystack in FluentCart - supports one-time payments, subscriptions, and automatic refunds.

== Description ==

Paystack for FluentCart seamlessly integrates Paystack payment gateway with FluentCart, allowing you to accept payments from customers in Nigeria, Ghana, South Africa, and globally.

= Features =

* **One-time Payments**: Accept card payments, bank transfers, USSD, and more
* **Subscriptions**: Support for recurring payments (coming soon)
* **Refunds**: Process refunds directly from FluentCart
* **Webhooks**: Automatic payment verification via webhooks
* **Test Mode**: Test your integration before going live
* **Secure**: All transactions are encrypted and secure

= Supported Currencies =

* NGN (Nigerian Naira)
* GHS (Ghanaian Cedi)
* ZAR (South African Rand)
* USD (US Dollar)

= Requirements =

* FluentCart plugin (free or pro)
* Paystack account ([Sign up here](https://paystack.com))

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/paystack-for-fluent-cart/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to FluentCart > Settings > Payment Methods
4. Enable and configure Paystack with your API keys
5. Save your settings

== Frequently Asked Questions ==

= Do I need a Paystack account? =

Yes, you need a Paystack account. You can sign up at https://paystack.com

= Where do I get my API keys? =

Log into your Paystack dashboard and navigate to Settings > API Keys & Webhooks

= Does this support subscriptions? =

Yes, subscription support is included and will be fully implemented based on Paystack's subscription API.

= Is it secure? =

Yes, all transactions are processed securely through Paystack's infrastructure. No card details are stored on your server.
